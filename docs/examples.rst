Examples
========


Build Examples
--------------

:: 

   cd examples
   mkdir buildMake
   cd buildMake
   cmake ..
   make



Examples
--------

- working directory is in the folder ``examples/``
- binary files are put in the folder ``examples/bin/``
- input files are put in the folder ``test_inputs/``
- The scripts to run the executable is put in ``examples/script/``
